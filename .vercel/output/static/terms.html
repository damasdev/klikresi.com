Ketentuan Layanan

Harap baca ketentuan layanan ini dengan seksama sebelum menggunakan situs web kami atau layanan kami. Dengan mengakses atau menggunakan situs web kami, Anda menyetujui dan mengikatkan diri pada ketentuan layanan ini. Jika Anda tidak setuju dengan ketentuan ini, kami sarankan agar Anda tidak menggunakan situs web atau layanan kami.

1. Penggunaan Situs Web

1.1. Konten Situs Web: Isi situs web kami, termasuk teks, gambar, audio, video, dan materi lainnya, adalah milik Klik Resi dan dilindungi oleh hak cipta dan hak kekayaan intelektual lainnya. Anda hanya diizinkan untuk menggunakan konten ini untuk keperluan pribadi dan non-komersial.

1.2. Akun Pengguna: Jika Anda membuat akun pengguna, Anda bertanggung jawab untuk menjaga keamanan akun Anda dan informasi login Anda. Anda juga harus memberikan informasi yang akurat saat mendaftar. Anda bertanggung jawab atas semua aktivitas yang terjadi di bawah akun Anda.

1.3. Penghentian Akses: Kami berhak untuk menghentikan atau membatasi akses Anda ke situs web kami kapan saja tanpa pemberitahuan sebelumnya jika Anda melanggar ketentuan layanan ini atau melakukan tindakan yang merugikan kami atau pengguna lain.

2. Layanan Kami

2.1. Deskripsi Layanan: Kami menyediakan Tracking API melalui situs web kami. Deskripsi layanan ini dapat berubah tanpa pemberitahuan sebelumnya.

2.2. Harga dan Pembayaran: Harga produk atau layanan kami akan dijelaskan di situs web kami. Anda setuju untuk membayar harga yang tercantum saat Anda melakukan pembelian. Kami dapat mengubah harga atau mengenakan biaya tambahan sesuai dengan kebijakan kami.

2.3. Pengiriman dan Pengembalian: Detail pengiriman, pengembalian, dan kebijakan garansi akan dijelaskan di situs web kami.

3. Hak dan Tanggung Jawab Anda

3.1. Ketentuan Penggunaan: Anda setuju untuk menggunakan situs web kami sesuai dengan hukum yang berlaku dan tidak melanggar hak kekayaan intelektual atau hak lainnya.

3.2. Konten Pengguna: Jika Anda mengirimkan konten pengguna ke situs web kami, Anda bertanggung jawab atas konten tersebut. Kami berhak untuk menghapus atau mengubah konten pengguna yang melanggar ketentuan ini atau yang dianggap tidak pantas.

4. Batasan Tanggung Jawab Kami

4.1. Kehilangan atau Kerusakan: Kami tidak bertanggung jawab atas kehilangan atau kerusakan yang timbul akibat penggunaan situs web atau layanan kami. Anda menggunakan situs web kami dengan risiko Anda sendiri.

4.2. Penghentian Layanan: Kami berhak untuk menghentikan layanan kami kapan saja tanpa pemberitahuan sebelumnya.

5. Perubahan Ketentuan Layanan

Kami dapat mengubah ketentuan layanan ini kapan saja dengan mengupdate halaman ini. Perubahan akan berlaku segera setelah diposting. Anda diharapkan untuk memeriksa halaman ini secara berkala untuk memahami ketentuan yang berlaku saat ini.

6. Kontak Kami

Jika Anda memiliki pertanyaan atau komentar tentang ketentuan layanan ini, silakan hubungi kami di [alamat email atau alamat fisik kontak].

Kamu bisa memonitor semua pengirimanmu secara real-time dengan berbagai fitur terintegrasi.
